Guesthouse Website

A modern and visually engaging webpage designed for a guesthouse.
Built with HTML, CSS, JavaScript, and Next.js to provide a smooth and attractive experience for potential guests.

✨ Project Highlights

🏡 Simple & clean design focused on showcasing the guesthouse

🖼️ Image-heavy layout to quickly grab attention and visually present what the guesthouse offers

📱 Fully responsive — works well on desktop, tablet, and mobile

🚫 No long paragraphs — people have short attention spans, so the content is concise and clear

🔧 Technical Features

Interactive and functional updates were added including:

✔️ Interactive elements using JavaScript
✔️ Backend setup using Node.js
✔️ Nodemailer integrated into a custom contact form
➡️ Form submissions are automatically sent directly to the guesthouse email

📬 Contact Form Integration

The contact form allows users to:

Send inquiries quickly

Get a seamless communication flow between customers and the guesthouse

Avoid needing long text — just the essentials to get in touch

🚀 Tech Stack
Category	Technology
Frontend	HTML, CSS, JavaScript, Next.js
Backend	Node.js, Nodemailer
💡 Goal

To create a fast, visually appealing platform that encourages visitors to explore the guesthouse through images first, supporting quick decision-making in today’s fast-paced digital world.
